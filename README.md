# Instalasi
## Termux
```bash
> apt update && apt upgrade
> pkg install libweb nodejs git ffmpeg imagemagick
> git clone https://github.com/ktdprjct/ktdprjctbot.git
> cd ktdprjctbot
> npm install
> node .
```
